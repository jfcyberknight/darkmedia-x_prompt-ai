Installer l'application

Pour installer DarkMedia Prompt AI sur votre appareil et l'utiliser comme une application native (hors ligne, plein écran, sans barre d'adresse) :

📱 Sur iPhone / iPad (Safari) :

  1. Appuyez sur le bouton de partage 📤 en bas de l'écran.
  2. Faites défiler vers le bas et sélectionnez « Sur l'écran d'accueil » ➕.

💻 Sur Chrome / Edge / Firefox :

Si vous êtes sur ordinateur ou Android, l'application est installable directement en cliquant sur le bouton « Installer l'application » dans la barre supérieure ou ci-dessous.
